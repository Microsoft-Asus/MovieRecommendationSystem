IMDB content data for movies in the EachMovie data set
------------------------------------------------------

Each file corresponds to the data for extracted from IMDB for the
movies in the EachMovie data set. The file name includes the movie id
from the EachMovie data set.

Each file contains the text information extract from each field in
IMDB. The name of the text field appears in bold before each
entry. These fields include TITLE, DIRECTOR, GENRE, etc.

Stopwords have been filtered out of the text. Proper nouns have been
converted to single tokens, e.g., woody_allen.

This data is being released to you as is. Please do not distribute
this data without approval from the authors.
